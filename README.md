# MACRO5000
- - -
![[Pasted image 20260313003803.png]]
This MacroPad features 10 MX Switches, a 0.96 inch screen, and a rotary encoder. Every part has its own purpose
- - -
# CAD
---
This HackPad Has A fun little aesthetic in where you can see the brains of the keyboard working to help you in your needs, it was made in SolidWorks. I used heatset inserts for 4x M3x16mm screws in a sandwich type of assembly.   Looks like the sky.

# Firmware
- - -
I used [QMK](https://qmk.fm/) . Have two layers, first is a normal numpad, the second is for macros that I would as I want. Encoder controls volume and the screen contains which layer I am on.
- - -
# BOM
---
- 1x  Seeed Studio XIAO RP2040
- 10x 1N4148 Diodes
- 10x MX-Style switches
- 1x EC11 Rotary encoders
- 1x 0.91 inch OLED display
- 10x white blank DSA keycaps
- 4x M3x16mm screws
- 4x M3x5mx4mm heatset inserts
- 1x Case (2 3d printed parts; top and bottom)
